# PhotoRec

### PhotoRec is a data recovery tool designed to recover files from storage devices where the file system has been damaged or formatted.

- Instead of relying on file names or directory structures, it operates by analyzing **file signatures within the data to extract files**.
- For this reason, file recovery is possible even when partitions are damaged or file system information has been lost.
- It is commonly used for recovering deleted files and for digital forensics analysis on various storage devices such as USB flash drives, SD cards, and external hard drives.

# PhotoRec

### PhotoRec은 **파일 시스템이 손상되었거나 포맷된 저장 장치에서 파일을 복구하기 위한 데이터 복구 도구**입니다.

- 파일 이름이나 디렉터리 구조를 기준으로 복구하지 않고, **파일 내부의 시그니처를 분석하여 파일을 추출하는 방식**으로 동작합니다.
- 이 때문에 파티션이 손상되었거나 파일 시스템 정보가 사라진 경우에도 파일 복구가 가능합니다.
- USB 메모리, SD 카드, 외장 하드 등 다양한 저장 장치에서 삭제된 파일 복구 및 디지털 포렌식 분석 목적으로 활용됩니다.
